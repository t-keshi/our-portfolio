# our-portfolio

## 概要

自己紹介、職務経歴、技術力などを伝えるための Web サイト

```shell
├── content

│   └── projects
├── src
│   ├── assets
│   ├── components
│   │   ├── templates
│   │   ├── ui
│   │   └── theme
│   ├── hooks
│   ├── styles
│   ├── static
│   ├── pages

├── .eslintignore
├── .eslintrc.js
├── .gitignore
├── .prettierrc
├── package.json
├── tsconfig.eslint.json
└── tsconfig.json
```

## スクリプト

```shell
yarn develop
yarn build
yarn lint lint
yarn lint:fix
```
